## 🗓️Blog site (react)

# 📖Project description
This is simple blog site application built using React.
it displays the articles with a title,date,and preview text

# 🔗Features
Reusable components(Article,Header,ArticleList)
Props used to pass data between components
Display a list of blog posts

# 🛠️Technologis used
React(vite)
Javascript
HTML& CSS
 # 📁How to run the project
 clone the repository:
git clone git@github.com:munirasheikh-rgb/components-props-lab.git 

 navigate to the folder:
cd components-props-lab

 install dependancies:
 npm install

 Run in the developmet server:
 npm run dev
 
 open in the browser
http://localhost:5173/
